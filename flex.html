Flexbox CSS con Tailwind

Aprende a crear layouts unidimensionales flexibles y alinear elementos con facilidad usando Flexbox.

¿Qué es Flexbox? 🤔

Flexbox (Flexible Box Layout) es un sistema de diseño unidimensional que te permite distribuir y alinear elementos dentro de un contenedor de forma flexible. Es perfecto para componentes y layouts simples.

💡 Clave: Flexbox trabaja en una dimensión a la vez: o en fila (horizontal) o en columna (vertical).

📌 Cuándo usar Flexbox: Menús de navegación, alineación de elementos, centrado vertical/horizontal, distribución de espacio entre items.

🧩 Conceptos Fundamentales

📦 Flex Container (Padre)

El elemento que tiene display: flex. Controla cómo se distribuyen sus hijos.

<div class="flex">

🎁 Flex Items (Hijos)

Los elementos directos dentro del flex container. Se alinean y distribuyen según las reglas del padre.

<div>Item 1</div>

Anatomía de Flexbox:

<div class="flex">  <!-- Flex Container -->
  <div>Item 1</div>  <!-- Flex Item -->
  <div>Item 2</div>  <!-- Flex Item -->
  <div>Item 3</div>  <!-- Flex Item -->
</div>

🎯 Demo 4: Centrado Perfecto

La combinación más famosa de Flexbox: centrado horizontal y vertical al mismo tiempo.

¡Estoy perfectamente centrado! 🎯

Código utilizado:

<div class="flex items-center justify-center h-64">
  <div>¡Estoy perfectamente centrado!</div>
</div>

🔑 Truco clave: items-center (vertical) + justify-center (horizontal) = Centrado perfecto

📋 Hoja de Referencia Rápida

🔄 Dirección

  • flex-row - Horizontal →
  • flex-col - Vertical ↓
  • flex-row-reverse - ←
  • flex-col-reverse - ↑

↔️ Justify (horizontal)

  • justify-start - Inicio
  • justify-center - Centro
  • justify-end - Final
  • justify-between - Espaciado

↕️ Align (vertical)

  • items-start - Arriba
  • items-center - Centro
  • items-end - Abajo
  • items-stretch - Estirado

📊 Flex

  • flex-1 - Crece igual
  • flex-auto - Automático
  • flex-none - Tamaño fijo
  • grow - Puede crecer

📏 Gap

  • gap-2 - 0.5rem
  • gap-4 - 1rem
  • gap-8 - 2rem
  • gap-x-4 gap-y-8 - Diferente

🔄 Wrap

  • flex-wrap - Sí ajustar
  • flex-nowrap - No ajustar
  • flex-wrap-reverse - Invertido
